# 爱啪啪思道

就是有多账号支持的 App Store 辅助工具。不能替代 App Store 但是够你用了。

## 👀 预览

![截图](../../../Resources/Screenshots/Screenshot-0.png)

## 🌟 主要功能

- [x] 支持多账户
- [x] 支持指定应用商店的地区
- [x] 搜索应用
- [x] 下载应用
- [x] 安装应用（无需越狱）
- [x] 分享应用安装包
- [x] 购买免费应用
- [x] 支持巨魔商店（如果你已经安装的话能省下不少折腾的时间）

由于技术原因，此构建版本搭载的安装器将在 2025/5/20 后过期。后续等待更新即可。其他功能不受影响
